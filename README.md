# Tutorial Rollup.js

Ce tutorial est basé sur l'article suivant [Tutorial Rollup](https://code.lengstorf.com/learn-rollup-js/).  

# Installation 

# Cloner le dépôt
git clone git@github.com:jlengstorf/learn-rollup.git

# Move into the repo
cd learn-rollup/

# Install dependencies
npm install

# Start the watcher
npm run watch
